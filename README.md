# 30 Days of Data Science

Repositori ini dibuat untuk mendokumentasikan progress belajar data science saya selama 30 hari secara konsisten. Fokus utama dari tantangan mandiri ini adalah melatih kemampuan eksplorasi data (EDA), visualisasi, dan pembersihan data menggunakan Python.

Di setiap folder, terdapat catatan singkat mengenai dataset yang dianalisis, visualisasi hasil, serta refleksi jujur mengenai kendala dan seberapa banyak AI (LLM) membantu dalam penulisan kodenya.
